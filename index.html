O famoso
HTML, CSS e JavaScript

  • HTML (Linguagem de Marcação de HiperTexto) é o bloco
    de construção mais básico da web. Define o significado
    e a estrutura do conteúdo da web.

  • CSS é uma linguagem baseada em regras. Você define
    regras especificando grupos de estilo que devem ser
    aplicados para elementos particulares ou grupos de
    elementos na sua página web

  • JavaScript é uma linguagem de programação que permite
    a você implementar itens complexos em páginas web
