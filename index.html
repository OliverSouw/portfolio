Mas afinal, o que estou aprendendo?

  • HTML é o bloco de construção mais básico da web. Define o significado e a estrutura do conteúdo da web.

  • CSS define regras especificando grupos de estilo que devem ser aplicados para elementos particulares ou grupos de elementos na sua página

  • JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web.
